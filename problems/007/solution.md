# 何切る問題の回答 - 2025年11月26日

## 問題の再掲

**難易度**: ★★★★★☆☆☆☆☆ (5/10)
**テーマ**: 押し引き（Push/Fold Decision）

**局面**: 東2局、自風南、ドラ表示牌🀁（南）、10巡目
**点数状況**: 自分24000点（3位）、下家31000点（1位）、対面26000点（2位）、上家19000点（4位）

**手牌14枚**:
```
🀈🀉🀙🀚🀛🀜🀝🀔🀕🀖🀃🀃🀃🀅
(2m3m1p2p3p4p5p5s6s7s北北北發)
```

**状況**: 下家（西家）が9巡目にリーチ宣言

**問題**: 下家のリーチに対して、この14枚の手牌から何を切りますか？

## 回答

**🀅（發）を切って様子を見るべきです。**

これは「様子見戦略」の典型的な場面です。完全に降りるのではなく、今巡は安全を確保しつつ、次巡で手が進めば攻めに転じる柔軟性を保ちます。

## 解説

### 手牌の分析

**14枚の手牌構成**:
```
手牌: 🀈🀉🀙🀚🀛🀜🀝🀔🀕🀖🀃🀃🀃🀅
数字: 2m3m1p2p3p4p5p5s6s7s北北北發
```

**構造分析**:
- **北北北** (🀃🀃🀃): 暗刻（役牌）
- **567s** (🀔🀕🀖): 順子
- **1234p** (🀙🀚🀛🀜🀝): 4連形 → 123p順子 + 45p両面待ち（最良の解釈）
- **23m** (🀈🀉): 辺張待ち（4m待ち）
- **發** (🀅): 孤立牌

**向聴数**: イーシャンテン（1-shanten）✓

**有効牌**（テンパイに直接向かう牌）:
- **🀊（4m）**: 234m順子が完成 → テンパイ
- **🀛（3p）**: 雀頭形成 → テンパイ
- **🀞（6p）**: 456p順子が完成 → テンパイ
- **合計3種類のみ**

**重要**: 發は有効牌ではなく、切るべき孤立牌です。發を切ることで、1m/2m/1p/2p/4p/5pなどをツモって雀頭形成→テンパイという道筋も残ります。

  - 10巡目で有効牌は3種類のみ
  - テンパイまで最低1巡、和了まで最低2巡必要
  - 残り巡目を考えると和了確率は非常に低い

### 押し引きの判断

**リスク vs リターンの比較**:

- **自分の和了期待値**: 極めて低い × 仮に和了しても1000-3000点 = **非常に小さい**
- **相手への放銃損失**: 3000-6000点規模（最悪8000点）= **大きい**

**結論**: 和了期待値 << 放銃損失 → **攻めは不適切**

**順位・点数状況の考慮**:

- 現在3位（24000点）
- 2位（26000点）との差: 2000点（小さい）
- 1位（31000点）との差: 7000点（中程度）
- 4位（19000点）との差: 5000点（安全圏）

**評価**: 2位との差が小さく、1位との差も中程度。この状況で無理に攻めて3位→4位に転落するリスクは避けるべき。ただし、完全に降りる必要はなく、様子を見ながら機会を待つのが適切。

### 各候補の検討

押し引き問題では、**3つの戦略**を検討します：

1. **様子見戦略**（安全牌を切りつつ柔軟性を保つ）← **最も一般的**
2. **完全降り戦略**（複数巡にわたり現物のみ切る）← 極端な状況のみ
3. **攻め戦略**（危険牌でも手を進める）← 勝負をかける状況のみ

#### 候補1: 🀅（發）を切る - 様子見戦略（最善）★推奨★

**戦略**: 様子見（安全牌を切りつつ柔軟性を保つ）

**メリット**:
- **孤立牌**で手の進行に影響が少ない
- 發を切ることで、1m/2m/1p/2p/4p/5pなどをツモって雀頭形成→テンパイの道筋が広がる
- **次巡で有効牌（4m/3p/6p）を引けば攻めに転じられる柔軟性**
- 比較的安全（下家の河には見えないが、孤立字牌なので中程度の安全性）

**デメリット**:
- 絶対安全（現物）ではない → 若干のリスクはある
- ただし、發を3枚持っている可能性は低い（役牌として早めに鳴くか切るため）

**安全性の評価**:
- 發は下家の河に見えていない
- しかし、發の暗刻を持っている可能性は極めて低い
- 發は孤立字牌として比較的安全な部類

**結論**:
**この状況での最善手。安全を確保しつつ和了可能性も残す。**

これは「完全降り」ではなく「様子見」です：
- 今巡: 發を切って安全を確保
- 次巡: 有効牌（4m/3p/6p）を引いた場合 → 攻めに転じることを検討
- 次巡: 手が進まない場合 → 様子見を継続、または1p（現物）で完全に降りる選択も

#### 候補2: 🀙（1p）を切る - 完全降り戦略（消極的）

**戦略**: 完全降り（現物を切って絶対安全を確保）

**メリット**:
- **下家の河に見える現物（🀙）= 絶対安全（100%安全、0%リスク）**
- 放銃リスクをゼロにできる

**デメリット**:
- **1pは有効牌の一つ** → 手の進行可能性を削る
  - 1pを切ると、1234pの4連形が234pになり、柔軟性が下がる
  - 3pをツモっても雀頭形成の選択肢が減る
- この状況はまだ「完全降り」を選ぶほど極端ではない
  - オーラスではない（東2局）
  - 順位は3位だが4位とは5000点差で安全圏
  - 手は1シャンテンでまだ可能性がある

**重要な違い**:
- 候補1（發切り）: **様子見** = 次巡で手が進めば攻められる
- 候補2（1p切り）: **完全降り** = 和了可能性を大きく削る

**結論**:
現物で安全だが、この状況で完全降りを選ぶのは消極的すぎる。有効牌を切るデメリットが大きい。様子見（發切り）の方が柔軟性が高い。

#### 候補3: 🀝（5p）を切る - 攻め戦略（最悪）

**戦略**: 攻める（手を進める牌を切る）

**メリット**:
- 手牌整理としては一理ある（5pを切ると123p + 4pの形になる）

**デメリット**:
- **下家は筒子を1枚も切っていない** → 筒子中心の手の可能性が極めて高い
- **5pは筒子の中張牌** → 放銃リスク極めて高い
- 筒子待ちの可能性が高い状況で筒子を切るのは自殺行為

**放銃リスク**: **極めて高い**

**結論**:
**3つの候補の中で最も危険。絶対に避けるべき。**

下家が筒子を全く切っていない状況で筒子を切るのは、わざわざ放銃しに行くようなものです。

### 最終結論

**🀅（發）を切って様子見戦略を取るのが最善です。**

**理由のまとめ**:

1. **和了期待値 << 放銃損失** → 攻めは不適切
2. **しかし完全降りするほど極端な状況ではない**（東2局、3位、イーシャンテン）
3. **様子見が最適**:
   - 今巡: 發（孤立牌、比較的安全）を切る
   - 次巡: 有効牌を引けば攻めに転じる柔軟性を保つ
   - 手が進まなければ様子見継続、または1p（現物）で完全降り
4. **發を切るメリット**:
   - 雀頭形成の可能性が広がる（1m/2m/1p/2p/4p/5pなどで雀頭）
   - 有効牌を残せる（4m/3p/6pの道筋を保つ）
   - 比較的安全（孤立字牌）

**次巡の判断指針**:
- **有効牌（4m/3p/6p）をツモ** → テンパイ形を見て、攻めるか降りるか再判断
- **1m/2m/1p/2p/4p/5pをツモ** → 雀頭形成、手が進んだら攻めを検討
- **危険牌をツモ** → 1p（現物）で完全降り、または他の安全牌を探す

**重要**: これは「完全降り」ではなく「様子見」です。柔軟性を保ちながら安全を確保する、押し引き問題における最も一般的な戦略です。

## 別解・補足

### 完全降り（1p切り）を選ぶべき状況

もし以下の条件が揃っていれば、1p（現物）切りで完全降りを選ぶべきです：

1. **オーラス（南4局）** AND
2. **2位と3位が僅差（1000-2000点差）** AND
3. **放銃すると順位が下がる** AND
4. **手が3シャンテン以上で和了の見込みがない**

しかし、本問は：
- 東2局（まだゲーム序盤）
- 2位と2000点差だが、4位とは5000点差で安全圏
- イーシャンテンでまだ和了可能性がある

→ 完全降りは消極的すぎる

### 現物（げんぶつ）の重要性

**現物の定義**: リーチ者が既に切った牌 = 絶対に当たらない（100%安全）

本問では：
- **🀙（1p）が現物**: 下家の河に見える → 絶対安全
- **發は現物ではない**: 下家の河に見えない → 若干のリスクあり

しかし、發は孤立字牌として比較的安全であり、手の進行可能性を考えると發切りが優位です。

もし發すら危険と感じる場合や、次巡で手が全く進まなかった場合は、1p（現物）切りで安全を確保する選択肢もあります。

## 学習ポイント

### 1. 押し引きの3つの戦略を理解する

押し引き問題では、単純な「攻める/降りる」の二択ではなく、**3つの戦略**があります：

1. **様子見**（比較的安全な牌を切りつつ柔軟性を保つ）← **最も一般的**
2. **完全降り**（現物のみ切って流局を目指す）← 極端な状況のみ
3. **攻め**（危険牌でも手を進める）← 勝負をかける状況のみ

**ほとんどの押し引き問題では「様子見」がデフォルト戦略**です。

### 2. 様子見と完全降りの違い

- **様子見**: 今巡は安全を確保するが、次巡で手が進めば攻めに転じる柔軟性を保つ
- **完全降り**: 今後の全巡で現物のみ切り、和了可能性を完全放棄

**重要**: イーシャンテンで有効牌がある場合、完全降りは消極的すぎます。様子見で柔軟性を保ちましょう。

### 3. 河の情報を活用した相手の手の推測

相手の河から以下の情報を読み取りましょう：

1. **ドラの可視性**: ドラが見えない → ドラ持ちの可能性高い
2. **役牌の可視性**: 白發中や風牌の可視性
3. **染め手の可能性**: 1色が全く切られていない → ホンイツ/チンイツの可能性
4. **リーチタイミング**: 早いリーチ（≤8巡目）→ 良形の可能性高い

本問では：
- ドラ（西）が見えない → ドラ1枚以上の可能性
- 筒子を全く切っていない → 筒子中心の手

→ 筒子（特に5p）は極めて危険と判断できます

### 4. 有効牌と孤立牌の区別

- **有効牌**: テンパイに直接向かう牌（本問: 4m/3p/6p）
- **孤立牌**: 切ることで雀頭形成の可能性が広がる牌（本問: 發）

孤立牌を切ることで、多くの牌で雀頭形成→テンパイの道筋が広がります。有効牌と孤立牌を正確に区別しましょう。

### 5. 点数状況と順位を常に意識する

押し引き判断では、和了期待値と放銃損失だけでなく、**順位と点差**も考慮します：

- 2位と僅差 → 無理に攻めて順位を下げるリスクは避ける
- 4位と余裕あり → ある程度の余裕がある
- オーラスではない → まだ挽回のチャンスあり

本問は「無理に攻める状況ではないが、完全に諦める状況でもない」という典型的な様子見のケースです。
