# 何切る問題の回答 - 2025年11月16日

**局面**: 東2局1本場、9巡目、自風は西

**点数状況**:
- 自分（西家）: 24000点（2位）
- 下家（南家・親）: 29000点（1位）※リーチ中
- 対面（北家）: 26000点（3位）
- 上家（東家）: 21000点（4位）

**手牌（14枚）**:
```
🀈🀉🀙🀚🀛🀜🀝🀖🀗🀘🀅🀅🀅🀆
(2m3m1p2p3p4p5p7s8s9s發發發白)
```

**ドラ表示牌**: 🀑 (2s) ※ドラは🀒 (3s)

**河（捨て牌）**:
- **自分**: 🀀🀁🀂🀃🀏🀡🀘🀌 (東南西北9m9p9s6m)
- **下家（リーチ）**: 🀓🀕🀗🀙🀎🀊🀆リ (4s6s8s1p8m4m白 + リーチ宣言牌)
- **対面**: 🀄🀐🀒🀔🀞🀍🀋 (中1s3s5s6p7m5m)
- **上家**: 🀀🀁🀘🀡🀏🀌🀎 (東南9s9p9m6m8m)

**問いかけ**: この局面で、あなたは何を切りますか？攻めるべきか、それとも安全を重視して降りるべきか、慎重に判断してください。

## 回答

**🀆（白）を切るべきです。**

これは**様子見戦略**を採用する局面です。白は下家（リーチ者）の河に見える**現物（絶対安全牌）**であり、この巡は安全を確保しつつ、次巡以降の状況次第で攻めに転じる柔軟性を保ちます。

### 押し引きの3つの戦略と判断

押し引き問題では、以下の3つの戦略を検討すべきです:

#### 1. 様子見戦略（中間戦略）← 本局面での最善手

**定義**: 比較的安全な牌を切りつつ、手を進める可能性を残す柔軟な戦略

**適用条件**（本局面は全て満たす）:
- ✓ 現物や安全牌がある（白が現物）
- ✓ イーシャンテンで有効牌が残っている（1m、4mの2種類）
- ✓ 非クリティカルな順位・点数状況（2位、オーラスではない）
- ✓ 相手リーチだが絶望的な状況ではない
- ✓ 巡目9（中盤）でまだ手に可能性がある

**この戦略の重要ポイント**:
- **これは「降りる」ではなく「柔軟性を保つ」戦略**
- 今巡は安全を確保するが、次巡で手が進めば攻めに転じる可能性を残す
- 完全に和了可能性を放棄するわけではない

#### 2. 完全に降りる（ベタ降り）← 本局面では不適切

**定義**: 複数巡にわたり現物のみを切り続け、全ての和了可能性を放棄

**適用条件**（本局面は満たさない）:
- ✗ オーラス（南4局）でクリティカルな順位 → 本局面は東2局
- ✗ 手が3シャンテン以上 → 本局面はイーシャンテン
- ✗ 複数の現物で残り巡目を全て切れる → 白は1枚のみ

本局面では、手がイーシャンテンで有効牌もあるため、完全降りは消極的すぎます。

#### 3. 全力で攻める ← 本局面では不適切

**定義**: 危険牌でも手を進める牌を切る

**適用条件**（本局面は満たさない）:
- ✗ 和了しないと順位が上がらない → 2位で安定、無理に攻める必要なし
- ✗ テンパイまたは良いイーシャンテン → イーシャンテンだが有効牌2種類のみ
- ✗ 高打点の可能性 → 發のみで1翻、低打点

本局面では、和了期待値が極めて低いため、全力で攻める理由はありません。

### 各候補の検討（戦略別分類）

#### 候補1: 🀆（白）を切る - 様子見戦略（最善）

**戦略**: 様子見（安全牌を切りつつ柔軟性を保つ）

**メリット**:
- **現物（絶対安全、100%安全）**: 下家の河に🀆が見える → 放銃リスク0%
- **孤立牌**: 手牌構成に影響が少ない
- **柔軟性を保つ**: 次巡で🀇 (1m)や🀊 (4m)を引けば、攻めに転じることを検討できる
- **順位を守る**: 放銃して3位転落を防ぐ

**デメリット**: 特になし

**結論**: **本局面での最善手。安全を確保しつつ、和了可能性も完全には放棄しない。**

#### 候補2: 🀙（1p）を切る - 攻める戦略（不適切）

**戦略**: 攻める（手を進めるが危険牌を切る）

**メリット**:
- 端牌（1p）で比較的安全な可能性
- 対面の河に🀙 (1p)が見えている（ただし対面はリーチしていない）

**デメリット**:
- **現物ではない** → 下家（リーチ者）に対して放銃リスクあり
- 下家の河に🀙 (1p)は1枚見えているが、待ちの可能性は否定できない
- **和了期待値が極めて低い**のに、わざわざリスクを取る理由がない
- 現物（白）がある以上、安全策を取るべき

**結論**: 現物（白）がある以上、わざわざリスクを取る必要はない。不適切。

#### 候補3: 🀝（5p）を切る - 攻める戦略（最悪）

**戦略**: 攻める（有効牌を切って手を進める）

**メリット**: なし（5pは有効牌の一部を構成する）

**デメリット**:
- **現物ではない** → 放銃リスクあり
- **下家は筒子を1枚も切っていない**（1pのみ） → 筒子染めまたは筒子中心の手の可能性
- **5pは中張牌**で、待ちになる可能性が高い
- **放銃リスク: 極めて高い**
- 有効な搭子（45p）を崩すことになり、手が遠のく

**結論**: **3つの候補の中で最も危険。絶対に避けるべき。**

#### 候補4: 🀅（發）を切る - 攻める戦略（非常に不適切）

**戦略**: 攻める（役牌を切って手を崩す）

**メリット**: なし

**デメリット**:
- **唯一の役（發の暗刻）を失う** → 和了しても役なし（ノーテン）
- **放銃リスクもある** → 發は役牌で、下家が持っている可能性あり
- 手を完全に崩すことになる

**結論**: **論外。絶対に切ってはいけない。**

### 最終的な比較と結論

**和了期待値 vs 放銃リスク**:
- **自分の和了期待値**: 極めて低い（イーシャンテンで有効牌2種類のみ、仮に和了しても1000-2000点程度）
- **放銃時の損失**: 3000-12000点規模（現実的には3000-6000点）
- **順位への影響**: 放銃すると3位転落の可能性が高い

**定性的な比較**:
```
和了期待値（極めて低い、仮に和了しても1000-2000点）
  <<
放銃時の損失（3000-6000点規模、順位転落のリスク）
```

**戦略選択**:
本局面は**様子見戦略**が最適です。白（現物）を切ることで:
1. この巡は完全に安全を確保（放銃リスク0%）
2. 次巡で🀇 (1m)や🀊 (4m)を引いた場合、状況を見て攻めに転じることを検討できる
3. 完全に和了可能性を放棄するわけではなく、柔軟性を保つ

**結論**: **🀆（白）を切るべきです。**

## 学習ポイント

### 1. 様子見戦略の重要性

押し引き問題では、「攻める」「降りる」の二択ではなく、**「様子見」という中間戦略が最も一般的**です。

**様子見戦略のポイント**:
- 現物や比較的安全な牌を切る
- 手を進める可能性を完全には放棄しない
- 次巡の状況次第で攻めに転じる柔軟性を保つ
- **完全降りとは異なる** → 和了可能性を残す

**適用場面**:
- イーシャンテンで有効牌が残っている
- 現物や安全牌がある
- 非クリティカルな順位・点数状況
- オーラスではない

### 2. 現物（げんぶつ）の価値

**現物 = リーチ者が既に捨てた牌 = 絶対安全（100%安全、放銃リスク0%）**

本局面では、🀆（白）が下家の河に見えるため、白は**現物**です。現物がある場合、それを活用することで放銃リスクを完全に回避できます。

**重要**: 「比較的安全」ではなく「絶対安全」であることを理解する。

### 3. 和了期待値と放銃リスクの比較

押し引き判断では、以下を比較します:
- **和了期待値** = 和了確率 × 和了時の点数
- **放銃リスク** = 放銃確率 × 放銃時の損失

本局面では:
- 和了期待値: 極めて低い（イーシャンテンで有効牌2種類、低打点）
- 放銃リスク: 高い（相手は親、早いリーチ、3000-6000点規模の損失）

**和了期待値 << 放銃リスク** → **様子見または降りが正解**

### 4. 順位・点数状況の考慮

麻雀は点数だけでなく順位も重要です。本局面では:
- 2位で安定している
- 3位（対面）との差は2000点のみ → 放銃すると3位転落の可能性
- 1位（下家）との差は5000点 → 和了しても追いつけない

**順位を守ることを優先** → 無理に攻めず、様子見が正解

### 5. 孤立牌の優先的な処理

白は孤立牌（他の牌と連携していない）であり、かつ現物です。孤立牌は手牌構成に影響が少ないため、優先的に切るべきです。

**孤立牌の優先順位**:
1. 現物の孤立牌（白） ← 最優先
2. 役牌でない孤立字牌
3. 端牌（1,9）の孤立牌

### 6. 相手の河の読み

下家の河から以下を読み取ります:
- 筒子を1pしか切っていない → 筒子中心の手の可能性
- 7巡目の早いリーチ → 良形待ちの可能性が高い
- ドラ3sが見えていないわけではないが、持っている可能性もある

河の情報を活用して、相手の手を推測し、危険牌を避けることが重要です。
