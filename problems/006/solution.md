# 何切る問題の回答 - 2025年11月24日

## 問題の再掲

**場**: 東2局、自風: 南家、ドラ表示牌: 🀒 (2s)、巡目: 11巡目

**点数状況**:
- 自分（南家）: 24000点（2位）
- 下家（西家）: 32000点（1位）
- 対面（北家）: 19000点（3位）←リーチ中
- 上家（東家）: 25000点（2位）

**手牌（13枚）**: 🀇🀈🀙🀚🀛🀜🀔🀕🀖🀃🀃🀃🀅 (1m2m1p2p3p4p5s6s7s北北北發)

**ツモ牌**: 🀝 (5p)

**問題**: この14枚から何を切るべきか？

## 回答

**🀅（發）を切る - 様子見戦略**

この局面では、現物である發を切って安全を確保しつつ、次巡で手が進んだ場合には攻めに転じる可能性を残す「様子見戦略」が最善です。

## 解説

### 手牌の分析

**14枚の手牌構成**:
```

🀔🀕🀖 (5s6s7s)
🀃🀃🀃 (北北北)

```

**手牌の構造**:
- **北北北**: 暗刻（役牌・自風）= 2翻確定
- **567s**: 完成した順子
- **12345p**: 5連続牌

- **12m**: 辺張（3m待ち）
- **發**: 孤立牌（切るべき牌）

**現在の向聴数**: イーシャンテン（1シャンテン）

**有効牌**: 🀉（3m）、🀛（3p）、🀞（6p）の3種類のみ

- 3p（4枚中2枚見えている = 残り2枚）: 123p順子強化 or 雀頭形成 → テンパイ
- 6p（4枚中0枚見えている = 残り4枚）: 456p順子完成 or 雀頭形成 → テンパイ

**手の評価**: イーシャンテンだが有効牌は3種類（合計9枚）のみ。11巡目で残り牌山は約50枚程度。和了までは距離がある。

### 現物（げんぶつ）の確認

**重要**: 對面（北家・リーチ者）の河を確認すると、🀅（發）が見えています。

これは**現物（げんぶつ）= 絶対安全牌（100%安全、0%リスク）**です。

リーチ宣言者が既に切った牌は、その人の待ち牌にはなり得ないため、發を切れば絶対に放銃しません。

### 点数状況と順位の考察

**現在の状況**:
- 自分: 24000点（2位）
- 1位（下家）: 32000点（+8000点差）
- 3位（対面）: 19000点（-5000点差）
- 4位該当なし（東家も2位タイ）

**順位への影響**:
- **トップとの差**: 8000点差 → 今から追いつくのは難しいが、トップを目指す必要性は低い
- **3位との差**: 5000点差 → 放銃すると3位に逆転される可能性がある（対面が3000-6000点程度和了すると危険）
- **この局の目標**: 2位を守ることが最優先。無理に攻めてトップを目指す局面ではない。

### 各候補の検討

#### 候補1: 🀅（發）を切る - 様子見戦略（最善）

**戦略**: 様子見（安全牌を切りつつ柔軟性を保つ）

**メリット**:
1. **絶対安全**: 對面（リーチ者）の河に見える現物 → 100%安全、0%の放銃リスク
2. **孤立牌**: 發は孤立した字牌で、手の進行にほとんど影響しない
3. **柔軟性**: 發を切ることで、次巡で有効牌（3m, 3p, 6p）を引いた場合、攻めに転じる選択肢が残る
4. **雀頭候補の拡大**: 發を切ることで、1m, 2m, 1p, 2p, 4p, 5p などを引いた際に雀頭形成の可能性が広がる
5. **順位保持**: 2位を守る戦略として適切

**デメリット**:
- なし（孤立牌で安全牌）

**戦略の詳細**:
- **これは完全降りではない**: 様子見は「今巡は安全を取るが、次巡で手が良くなれば攻める」という柔軟な戦略
- 次巡のシナリオ:
  - 有効牌（3m, 3p, 6p）を引く → テンパイ、攻めに転じることを検討
  - 雀頭候補（1m, 2m, 1p, 2p, 4p, 5p）を引く → 手の選択肢が広がる、状況次第で攻める
  - その他の牌 → 引き続き安全牌を切って様子見継続
- **完全降りとの違い**: 完全降りは「残り全巡で現物だけを切り、和了可能性を完全放棄」するが、様子見は「今巡は安全だが、次で変わる可能性を残す」

**結論**: **この局面での最善手。安全を確保しつつ和了可能性も残す。**

#### 候補2: 🀙（1p）を切る - 攻める戦略（不適切）

**戦略**: 攻める（手を進める牌を残し、端牌を切る）

**メリット**:
- 端牌で比較的安全と思われる（下家の河に1pが1枚見える）
- 12345pの連続牌を整理できる

**デメリット**:
1. **現物ではない** → 放銃リスクあり
2. **對面は筒子をほとんど切っていない**（1pを1枚切っているのみ） → 筒子待ちの可能性が高い
3. **1pは端牌だが安全とは限らない**: 早巡に切られた1pは比較的安全だが、8巡目リーチの相手に対しては危険
4. **有効牌を1種類減らす**: 1pを切ると、1p2pで雀頭を作る選択肢がなくなる
5. **現物（發）がある以上、わざわざリスクを取る必要がない**

**リスク評価**:
- 對面は1pを切っているため「1p単騎」の可能性は低いが、12p、14pの辺張・嵌張待ちの可能性はある
- 放銃率: 低〜中程度（現物ではないため0%ではない）

**結論**: 現物（發）がある状況で、わざわざリスクを取る必要はない。不適切な選択。

#### 候補3: 🀝（5p）を切る - 攻める戦略（最悪）

**戦略**: 攻める（ツモ牌を切り、手牌の12345pを整理する）

**メリット**:
- 5pを切ることで、1234pの4連形が残り、将来的に待ち選択の余地が生まれる可能性がある

**デメリット**:
1. **對面は筒子をほとんど切っていない**（1pを1枚のみ） → 筒子待ちの可能性が極めて高い
2. **5pは中張牌で最も危険な牌の一つ** → 345p, 456p, 567pの待ちに刺さる可能性が高い
3. **放銃リスク: 極めて高い** → 對面の河に筒子がほとんど見えていない以上、筒子の中張牌は最も危険
4. **有効牌を減らす**: 5pを切ると、45pの両面搭子が崩れ、手の進行が遅くなる
5. **現物（發）がある以上、絶対に避けるべき**

**リスク評価**:
- 對面の河: 1pのみ（筒子） → 筒子待ちの可能性が極めて高い
- 5pは筒子の中で最も多くの待ちに刺さる牌（345p, 456p, 567pの3種類の順子待ち）
- 放銃率: 極めて高い

**結論**: **3つの候補の中で最も危険。絶対に避けるべき。**

#### 候補4: 🀉（3m）を切る - 攻める戦略（論外）

**戦略**: 攻める（有効牌を切って安全を取る）

これは有効牌を自ら切る選択で、問題文の趣旨から外れるため詳細は省略しますが、参考までに触れます。

**デメリット**:
- 3mは有効牌（12mを123mに変える唯一の牌）
- 3mを切ると、12mは永遠に浮き牌のまま
- 手の進行を自ら止める行為

**結論**: 論外。有効牌を切る選択は避けるべき。

### 期待値の比較

**自分の和了期待値**:
- 和了可能性: 低い（イーシャンテン、有効牌3種9枚、11巡目、残り牌山約50枚）
- 和了時の打点: 2900-7700点（現実的には3900-7700点）
- 定性評価: 和了期待値は「低い」

**對面への放銃リスク**:
- 發を切る場合: 0%（現物）
- 1pを切る場合: 低〜中程度（現物ではないが端牌）
- 5pを切る場合: 極めて高い（筒子中張牌、對面は筒子をほとんど切っていない）
- 放銃時の損失: 3000-6000点規模（最悪8000点）

**比較**:
- **様子見戦略（發を切る）**: 放銃リスク0%、和了可能性を残す → **最もバランスが良い**
- **攻める戦略（1pや5pを切る）**: 放銃リスク中〜高、和了可能性は低いまま → **リスクとリターンが釣り合わない**

**定性的比較**:
```
和了期待値（低い、仮に和了しても3900-7700点程度） << 放銃損失（3000-6000点規模、リスク高）
```

現在2位で、3位との差が5000点しかない状況では、放銃すると3位転落の危険があります。一方、和了してもトップに追いつくのは難しい（8000点差）。したがって、**2位を守る戦略が最優先**であり、様子見戦略が最適です。

### 結論

**🀅（發）を切る様子見戦略が最善**

**理由**:
1. **絶対安全**: 現物であり、放銃リスク0%
2. **柔軟性**: 次巡で手が進めば攻めに転じる選択肢を残す
3. **順位保持**: 2位を守る戦略として適切（放銃して3位転落を避ける）
4. **孤立牌**: 手の進行にほとんど影響しない

**様子見の重要性**:
- これは「完全に降りる」ではなく、「今巡は安全を取るが、次巡で攻めのチャンスがあれば転じる」という柔軟な戦略
- 押し引き問題の多くは、この「様子見」がデフォルトの正解となる
- 完全降り（ベタ降り）は、オーラスでクリティカルな順位状況など、極端な場合にのみ使う

## 別解・補足

### 完全降り（ベタ降り）戦略は適切か？

問題文では「ベタ降り（完全降り）」も選択肢として挙げられていますが、この局面では**完全降りは不要**です。

**完全降りが適切な状況**:
- オーラス（南4局）でクリティカルな順位（例: 2位だが3位と僅差で、3位転落が致命的）
- 手が3シャンテン以上で和了の見込みが全くない
- 複数の現物があり、残り巡目を全て現物で切れる

**現在の状況**:
- 東2局（まだゲーム中盤）
- イーシャンテン（有効牌3種9枚）
- 2位と3位の差は5000点（大差ではない）

→ **この状況では完全降りは過剰防御**。様子見で十分です。

### 攻める戦略は全くないのか？

仮にトップを追いたい場合、攻める選択肢もありますが、以下の理由から不適切です：

1. **手の遠さ**: イーシャンテン、11巡目、有効牌3種9枚のみ → 和了可能性は低い
2. **打点の期待値**: 仮に和了しても3900-7700点程度 → トップとの8000点差は埋まらない
3. **放銃リスク**: 筒子を切ると放銃の可能性が高い → 3位転落の危険
4. **順位状況**: 2位を守ることが優先 → トップを狙う局面ではない

したがって、この局面で攻める選択は**リスクとリターンが釣り合わない**ため、不適切です。

### もし發が現物でなかったら？

仮に發が現物でなかった場合（對面が發を切っていなかった場合）、以下のような判断になります：

1. **字牌の安全度**: 發は役牌ではないため、比較的安全
2. **他の安全牌**: 1m, 2m などの端牌を切る選択肢も
3. **筋読み**: 對面の河を見て、筋の安全な牌を選ぶ

ただし、實際には發は現物なので、迷わず發を切るべきです。

## 学習ポイント

### 1. 押し引きの3つの戦略を理解する

押し引き問題では、単純な「攻める/降りる」の二択ではなく、**3つの戦略**があります：

1. **様子見（中間戦略）** ← **最も一般的なデフォルト戦略**
   - 比較的安全な牌を切りつつ、手の進行可能性を残す
   - 柔軟性を維持し、次巡で状況が変われば戦略を変える
   - **多くの押し引き問題ではこれが正解**

2. **完全降り（ベタ降り）** ← 極端な状況でのみ
   - 現物のみを切り続け、和了可能性を完全放棄
   - オーラスでクリティカルな順位状況などでのみ使用

3. **全力で攻める** ← 極端な状況でのみ
   - 危険牌でも手を進める
   - 和了しないと順位が上がらない状況でのみ使用

**重要**: 多くの押し引き問題では、**様子見がデフォルトの正解**です。完全降りや全力攻めは極端な状況でのみ使います。

### 2. 現物（げんぶつ）の価値を理解する

**現物 = リーチ者が既に切った牌 = 絶対安全牌（100%安全、0%リスク）**

- 現物がある場合、迷わず現物を切るのが基本
- 「比較的安全」ではなく「絶対安全」であることを理解する
- 現物を見逃して危険牌を切るのは致命的なミス

この問題では、發が現物であることに気づくことが最重要ポイントです。

### 3. 相手の河を読む習慣をつける

對面の河を分析することで、相手の手の強さを推測できます：

- **ドラの可視性**: ドラが見えていない → ドラ持ちの可能性が高い
- **役牌の可視性**: 役牌が見えていない → 役牌持ちの可能性
- **染め手の可能性**: 切られている色を確認
- **リーチタイミング**: 早いリーチ → 良形の可能性が高い

この問題では、ドラ🀒が見えていないため、對面は少なくとも2翻以上（リーチ+ドラ）の可能性が高いと推測できます。

### 4. 順位と点数状況を考慮する

麻雀は点数だけでなく、**順位**も重要です：

- トップとの差が大きい → トップを追うのは難しい、現在の順位を守る
- 下位との差が小さい → 放銃すると順位が下がる危険
- オーラスか中盤か → オーラスはより慎重に、中盤はまだ挽回の余地あり

この問題では、2位を守ることが最優先であり、無理に攻めてトップを狙う局面ではありません。

### 5. 孤立牌の処理

孤立した字牌（發）は、手の進行にほとんど影響しない「価値の低い牌」です。

- 孤立字牌は優先的に切る
- 特に現物である場合、最優先で切るべき
- 孤立字牌を切ることで、他の牌で雀頭を作る選択肢が広がる

この問題では、發を切ることで、1m, 2m, 1p, 2p, 4p, 5p などで雀頭を作る可能性が広がります。

### 6. 期待値思考の重要性

押し引き判断では、**期待値**を考えることが重要です：

- **和了期待値**: 和了可能性 × 和了時の打点
- **放銃リスク**: 放銃可能性 × 放銃時の失点
- **比較**: 和了期待値 vs 放銃リスク

この問題では：
- 和了期待値: 低い（イーシャンテン、有効牌少ない、11巡目）
- 放銃リスク: 筒子を切ると高い、發（現物）なら0%

→ **發を切る様子見戦略が最もバランスが良い**

---

**まとめ**: この問題は、現物を見つけて様子見戦略を選択することが鍵でした。押し引き問題の基本である「様子見」の重要性を理解し、状況に応じて柔軟に戦略を変える思考を身につけましょう。
