# 何切る問題の回答 - 2025年11月16日

## 問題の再掲

**局面情報**:
- 場: 東2局1本場
- 自風: 南家（子）
- ドラ表示牌: 🀑 (2s)
- 巡目: 11巡目
- 点数状況: 自分23000点（2位）、下家28000点（1位）、対面26000点、上家23000点（3位）

**手牌14枚（13枚+ツモ）**:
```
🀇🀈🀙🀛🀜🀝🀞🀔🀕🀖🀃🀃🀃🀅
(1m2m1p3p4p5p6p5s6s7s北北北發)
```

**状況**: 8巡目に上家がリーチを宣言。11巡目の現在、攻めるべきか降りるべきかの判断が求められる。

**問題**: この局面で何を切りますか？攻めるべきか、降りるべきか判断してください。

---

## 回答

**🀅（發）を切って降りるべきです。**

この局面では完全に降りて、現物である發を切り、以降も安全牌を優先して流局を目指すのが最善です。

---

## 解説

### 手牌の分析

**手牌14枚の構成**:
```
🀇🀈 (1m2m) - 辺張待ち（3mで順子完成）
🀙🀛🀜🀝🀞 (1p3p4p5p6p) - 散らばっている
🀔🀕🀖 (5s6s7s) - 完成した順子 ✓
🀃🀃🀃 (北北北) - 完成した暗刻 ✓ 役牌
🀅 (發) - 孤立牌
```

**向聴数**: イーシャンテン（1シャンテン）
- 完成面子: 2つ（北北北 + 567s）
- 不完全面子: 1つ（12m辺張）
- 有効牌: 🀉 (3m) を引けば123mが完成しテンパイに近づく

**現在の形の問題点**:
- 11巡目でイーシャンテン → 残り約7巡で和了まで到達するのは困難
- 有効牌の種類が限定的（主に3m）
- 北の暗刻があるため役はあるが、1翻のみで打点が低い

---

### 見えている牌の集計

**ドラ（🀓 3s）の可視性**:
- ドラ表示牌: 🀑 (2s) → ドラは 🀒 (3s)
- 対面の河に🀑が1枚 → ドラ表示牌1枚 + 河1枚 = 2枚確認
- **残り2枚の行方は不明** → 上家（リーチ者）が持っている可能性が高い

**自分の手牌からの残り枚数**（重要な牌のみ）:
- 🀉 (3m): 上家の河に1枚見える → 残り3枚
- 🀙 (1p): 対面の河に1枚 + 自分のツモ1枚 = 残り2枚
- 🀅 (發): 上家の河に1枚 + 自分の手に1枚 = 残り2枚

---

### 上家（リーチ者）の手の分析

**河からの読み**:
- 上家の河: 🀀🀁🀃🀅🀆🀋🀊🀉🀗🀖🀔 (東南北發白5m4m3m8s7s5s)
- **8巡目の早いリーチ** → 良形待ちの可能性が高い
- **字牌を全て切っている** → 役牌での役はなし
- **萬子**: 3m, 4m, 5mを切っている → 萬子は使っていない
- **索子**: 5s, 7s, 8sを切っている → 索子は一部のみ使用
- **筒子**: 1枚も切っていない → **筒子中心の手と推測**

**ドラの可視性**:
- ドラ🀒 (3s) は2枚しか見えていない（表示牌 + 対面の河の🀑 1枚）
- 残り2枚は不明 → **上家が1枚以上持っている可能性が高い**

**上家の想定手役パターン**:
1. リーチのみ = 1翻 = 1000-2000点（ドラが見えていないため可能性は低い）
2. リーチ(1) + ツモ(1) = 2翻 = 2000-3900点
3. リーチ(1) + ドラ(1) = 2翻 = 2000-3900点
4. リーチ(1) + ピンフ(1) = 2翻 = 2000-3900点（筒子中心なら可能性あり）
5. リーチ(1) + ドラ(1) + ツモ(1) = 3翻 = 3900-5800点
6. リーチ(1) + ピンフ(1) + ドラ(1) = 3翻 = 3900-5800点
7. リーチ(1) + ピンフ(1) + ドラ(1) + ツモ(1) = 4翻 = 8000点（満貫）

**結論**: ドラが見えていないこと、筒子を1枚も切っていないことから、**2-3翻（2000-5800点）以上の可能性が高い**。最低でも2翻2000点以上、最悪の場合満貫8000点の可能性もある。

---

### 自分の手の期待値分析

**理想形と点数**:
- **最低ライン**: 北のみ(1翻) = 1000-2000点
- **リーチ込み**: リーチ(1) + 北(1) = 2翻 = 2000-3900点
- **ドラ込み**: リーチ(1) + 北(1) + ドラ(1、3sを引いた場合) = 3翻 = 3900-5800点
- **最良ケース**: リーチ(1) + ツモ(1) + 北(1) + ドラ(1) = 4翻 = 8000点（満貫）

**現実的な評価**:
- 現在11巡目でイーシャンテン → 残り約7巡
- 有効牌は主に🀉 (3m) のみ → 残り3枚
- テンパイまで到達する可能性: **低い**
- テンパイしても和了できる可能性: **非常に低い**
- 仮に和了しても1000-3900点程度

**和了期待値**: 非常に低い（遠い手、限定的な有効牌、残り巡目が少ない）

---

### 押し引き判断

**自分の期待値 vs 放銃リスク**:
- **自分の和了期待値**: 非常に低い × 1000-3900点程度 = **極めて低い**
- **放銃時の損失**: 2000-8000点規模（現実的には3000-6000点程度）
- **比較**: 和了期待値（極めて低い）<< 放銃損失（3000-6000点規模）

**順位・点数状況**:
- 自分: 2位 23000点
- 1位: 下家 28000点（差: -5000点）
- 3位: 上家 23000点（同点）
- **放銃した場合**:
  - 3000点放銃 → 20000点で3位転落の可能性
  - 6000点放銃 → 17000点で3位転落確定
- **流局した場合**: 2位維持

**現物の有無**:
- **🀅（發）は上家の河に見える** → **現物（絶対安全牌、100%安全）**
- 現物を持っているため、安全に降りることが可能

**結論**:
- 和了期待値が極めて低い
- 放銃リスクが大きい（2位から3位転落の危険）
- 現物（發）を持っている
- **→ 完全に降りるべき**

---

### 各候補の検討

#### 候補1: 🀅（發）を切る - 最善の選択（推奨）

**戦略**: 完全に降りる（ベタ降り）

**メリット**:
- **上家の河に🀅が見える → 現物（絶対安全牌、100%安全）**
- 放銃リスク: **0%**
- 孤立牌で手牌の進行にほぼ影響しない
- 役にならない牌（發は役牌だが自風・場風ではない）

**デメリット**:
- 和了の可能性を完全に放棄することになる
- しかし元々和了期待値は極めて低いため、実質的なデメリットは小さい

**以降の方針**:
- 引き続き安全牌を探して切る
- 次巡以降も現物や筋牌、字牌などの比較的安全な牌を優先
- 流局を目指して2位を守る

**結論**: **この状況での最善手。放銃を完全に回避しつつ2位を守る。**

---

#### 候補2: 🀙（1p）を切る - 次善の選択

**戦略**: 様子見（中間戦略）

**メリット**:
- 端牌（1p）は比較的安全
- 対面の河に🀙が1枚見えているため、やや安全性が高い
- 手牌の進行可能性を少し残せる

**デメリット**:
- **現物ではない** → 放銃リスクがゼロではない
- 筒子を1枚も切っていない上家に対して、1pは危険牌の可能性がある
- 上家は筒子中心の手と推測されるため、1pは待ち牌になりうる

**放銃リスク**: 中程度（筒子中心の手に対して端牌でも危険）

**結論**: 現物（發）がある以上、わざわざリスクを取る必要はない。発を切る方が明らかに優れている。

---

#### 候補3: 🀞（6p）を切る - 最悪の選択

**戦略**: 攻める（手を進める）

**メリット**:
- 3456pの形を整理できる
- 孤立した6pを切ることで、345pの順子と1pが残る

**デメリット**:
- **上家は筒子を1枚も切っていない** → **筒子待ちの可能性が極めて高い**
- **6pは上家の待ち牌である可能性が非常に高い**
- 放銃リスク: **極めて高い**
- 手を進めても和了期待値は依然として低い

**放銃リスク**: 極めて高い（上家の河に筒子が1枚もない）

**結論**: **3つの候補の中で最も危険。絶対に避けるべき選択。** 筒子待ちの可能性が高い上家に対して筒子を切るのは自殺行為に等しい。

---

## 結論

**🀅（發）を切って完全に降りるべきです。**

**理由のまとめ**:
1. **現物（絶対安全牌）を持っている**: 發は上家の河に見える → 100%安全
2. **和了期待値が極めて低い**: 11巡目イーシャンテン、有効牌限定、残り巡目少ない
3. **放銃リスクが大きい**: 3000-6000点規模の放銃で2位→3位転落の危険
4. **2位を守ることが優先**: 5000点差で1位を追うよりも、3位転落を防ぐことが重要

発を切った後も、引き続き安全牌を優先して打ち、流局を目指すことで2位を守りましょう。

## 別解・補足

### もし現物がなかったら？

仮に發が上家の河になく、現物がない場合でも、この局面では以下の順序で安全牌を探します:

1. **字牌（役牌以外）**: 孤立した字牌は比較的安全
2. **筋牌**: 早巡に切られた牌の筋（例: 上家が3mを早巡に切っているので1m-4mの筋）
3. **端牌**: 1, 9は中張牌より安全

この問題では幸い現物（發）があるため、迷わず切るべきです。

### 親番だったら？

もし自分が親番（東家）だった場合:
- 親の連荘を狙う価値があるため、やや攻める判断もありうる
- しかしこの手は11巡目イーシャンテンと遠いため、親番でも降りるのが妥当

### オーラス（南4局）だったら？

オーラスで2位の場合:
- 2位を守ることが最優先 → より確実に降りる
- この問題と同様、發を切って完全に降りるべき

## 学習ポイント

### 1. 現物（げんぶつ）の重要性

**現物とは**: リーチ者が既に捨てた牌 = 絶対に当たらない = 100%安全

- リーチがかかったら、まず相手の河を確認
- 手牌に現物があれば、それを優先して切る
- 「比較的安全」と「絶対安全（現物）」は全く異なる
- 現物は放銃リスクを完全にゼロにできる唯一の手段

### 2. 押し引き判断の3つの選択肢

押し引きは単純な「攻める/降りる」の二択ではなく、3つのグラデーションがある:

1. **完全に降りる（ベタ降り）**: 現物のみ切って流局狙い
2. **様子見**: 比較的安全な牌を切りつつ手の進行可能性を残す
3. **全力で攻める**: 危険牌でも手を進める牌を切る

この問題では、現物があり和了期待値が極めて低いため、**完全に降りる**が最適解でした。

### 3. 河からの読み（相手の手の推測）

上家の河を分析することで手の傾向を推測できる:
- **筒子を1枚も切っていない** → 筒子中心の手
- **字牌を全て切っている** → 役牌なし
- **ドラが見えていない** → ドラ持ちの可能性が高い
- **8巡目の早いリーチ** → 良形待ちの可能性が高い

これらの情報から、相手の待ち牌や点数を推測し、放銃リスクを評価します。

### 4. 順位と点差を考慮した判断

- 2位で同点の3位がいる状況 → 3位転落を防ぐことが最優先
- 1位との差は5000点あるが、この手では追いつけない
- **2位を守る > 1位を狙う** という判断

麻雀は点数だけでなく順位も重要です。無理に攻めて3位に落ちるよりも、堅実に2位を守る判断が求められます。

### 5. イーシャンテンの和了期待値

11巡目でイーシャンテンの場合:
- 残り約7巡で、テンパイ → 和了まで到達するのは困難
- 有効牌の種類が限定的（この問題では主に3mのみ）
- 仮に和了しても低打点（1-2翻程度）

→ **和了期待値は極めて低い** と判断するのが妥当

---

**この問題を通じて、「現物の価値」「押し引き判断の基準」「河からの読み」という3つの重要な技術を学ぶことができます。**
